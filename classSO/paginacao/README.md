# Simulador de Memória - Paginação

### Implemente um simulador que permita pelo menos as operações seguintes:
- Definição de tamanho de páginas;
- Definição de tamanho de memória lógica e física;
- Visualizar o mapeamento entre páginas e quadros;
- Permita visualizar os bits usados para diferenciar cada página;
- Permita associar um endereço dentro de uma página a um endereço de um quadro específico na memória (tradução).


## Iniciar o simulador
- Apenas abrir o código no VSCode;
- O prório VSCode ja roda facilmente o código para web;
- Clicar no botão edireito do mouse em cima do arquivo "paginacao.html" e clicar em "Open in Integrated Browser";
- Se preferir, pode baixar algum plugin para abrir em outro browser's como o "live server".

## Obervações
- O simulador gera os quadros/frames e paginações de acordo com o valor de memória digitados;
- Os quadros são escolhidos aleatoriamente, mas dentro do intervalo que foram criados de acordo com a quantidade de memória real.
